# WhatsApp Web Clone

A WhatsApp Web–like interface that displays real-time conversations using webhook payloads. Messages can be viewed, grouped by user, and demo messages can be sent (stored only, not actually sent).

## 🚀 Live Demo

- **Frontend**: https://whatsapp-web-clone-frontend.vercel.app  
- **Backend**: https://whatsapp-web-backend.onrender.com  

## ✨ Features

- WhatsApp-style chat UI
- Webhook payload processor (insert/update in MongoDB)
- Real-time updates using Socket.IO
- Message status tracking (sent/delivered/read)
- Responsive design (mobile + desktop)
- Demo message sending (stored in DB)

## 🏗️ Tech Stack

- **Frontend**: React, Socket.IO, Date-fns
- **Backend**: Node.js, Express, MongoDB Atlas, Socket.IO
- **Deployment**: Vercel (frontend), Render (backend)
